
# 📺 Instagram Automated Reels Scroller
A lightweight, "set-and-forget" JavaScript utility for the Instagram web interface. This script automates the viewing experience by clicking the "Next" button at a defined interval, perfect for a hands-free Reels experience.
✨ Features
 * Reliable Navigation: Targets the aria-label for "Next" to ensure it works even if Instagram's CSS classes change.
 * Customizable Pace: Set your own delay to match the length of the content you're watching.
 * Zero Dependencies: Pure Vanilla JavaScript; no extensions or libraries required.
 * Easy Kill-Switch: Simple command to stop the automation instantly.
🚀 How to Use
 * Open Instagram: Log in to instagram.com on your desktop browser.
 * Go to Reels: Navigate to the Reels tab.
 * Open Developer Tools: Right-click anywhere and select Inspect (or press F12) and click the Console tab.
 * Inject the Script: Copy the code from script.js, paste it into the console, and hit Enter.
> [!TIP]
> Adjusting Speed: The script defaults to 25 seconds (25000ms). To change this, edit the number at the bottom of the script. For 10 seconds, use 10000.
