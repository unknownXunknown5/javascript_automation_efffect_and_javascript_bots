<h1> Color Splash Particle Effect\</h1>

A creative JavaScript experiment that generates a vibrant "color splash" particle effect following the user's cursor movement.

🚀 Features
 * Dynamic Particle Generation: Particles spawn exactly at the cursor's X and Y coordinates.
 * Color Variance: Randomized colors for a "splash" aesthetic.
 * Smooth Animation: Optimized frame updates for fluid motion.
📂 Project Structure
 * index.html: The canvas container.
 * script.js: The logic for particle physics and cursor tracking.
 *  Basic styling to ensure a full-screen experience.
🛠️ How to Run
 * Clone the repository:
   git clone https://github.com/kriyanshkirito/javascript_bots.git

 * Navigate to the ColorSpalshParticle folder.
 * Open index.html in any modern web browser.
