# 🎠 Auto Image Carousel (HTML, CSS, JavaScript)

A simple and responsive **automatic image carousel (slider)** built using pure **HTML, CSS, and JavaScript**.
It automatically slides images at a fixed interval and also supports manual navigation.

---

## 🚀 Features

* ✅ Auto-sliding images
* ✅ Smooth transition animation
* ✅ Responsive design
* ✅ Lightweight (no external libraries)

## 🛠️ Customization

* Change slide interval:

```javascript
setInterval(nextSlide, 5000); // 5 seconds
```

* Add more images:

```html
<img src="images/new.jpg" class="slide">
```
